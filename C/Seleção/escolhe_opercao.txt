Crie um programa que recebe um número inteiro OP
e dois números reais X e Y e realiza uma determinada 
operação matemática com base no inteiro recebido.

O programa deve obedecer a seguinte lógica:

OP = 1    -> Somar X e Y
OP = 2    -> Subtrar Y de X
OP = 3    -> Multiplicar X e Y
OP = 4    -> Dividir X por Y

A saída do programa deve consistir apenas do resultado
esperado para a operação selecionada.

EXEMPLO:

>> 1
>> 2
>> 3
>> 5
FIM
>> 2
>> 2
>> 3
>> -1
FIM
